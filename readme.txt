very-simple-slider is one of simple image slider made for wordpress blog or site. 

== Description ==

"very-simple-slider" you can download it for you next image slider into wordpress site.

**Detailed features:** 

* Use it as your home page slider, into post or page.
* User friendly.
* You can use it into post and page by shortcode <code>[very_simple_slider id='page or post id' /] </code> into page or page and into template file using <code> <?php echo do_shortcode("[simple_slider_wp id='page or post id' /]"); ?></code>


== Installation ==

1. Upload the entire `very-simple-slider` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

You will find 'Very Simple Slider' menu in your WordPress admin panel.
After adding new slider, see it at All Sliders.
You will find the Shortcode for every slider.
